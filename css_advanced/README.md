CSS Advanced Project

This project is part of the ALX Front-End curriculum.
It focuses on structuring a complete, multi-section HTML page while applying advanced HTML concepts such as:

Semantic HTML structure

Proper nesting

Accessibility considerations

Media elements

Reusable components

Navigation layout

Multi-section landing page styling preparation

The design used in this project is based on the "SmileSchool" concept page.
